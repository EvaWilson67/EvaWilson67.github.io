CSCE 242

Web Applications is a course designed to help students not only be able to create the front end of a website with HTML and CSS, but also the backend of the website with javascript. Additionally, students will learn to utilize node and server side JavaScript to store and retrieve data.

Assignments

Assignment 1: Basic HTML

A demonstation on how to build a basic HTML website, I chose to build the website around my cats.

Assignment 2: Basic CSS

A demonstation on how to build a HTML website with CSS, I chose to build the website around kombucha.

Assignment 3: Page Layout

A demonstation on format and align a web page using CSS flex.

Assignment 4: Recreate CSS Page

Recreating USC College of Arts and Sciences page with Flex Boxes

Assignment 5: JavaScript Buttons, Functions, and More

A demonstation on how to use javascript buttons, functions, and more

Assignment 6: JavaScript: Conditionals

A demonstation on how to use javascript conditionals

Assignment 7: JavaScript: Loops

A demonstation on how to use javascript loops

Assignment 8: JavaScript: Arrays

A demonstation on how to use javascript arrays

Assignment 9: JavaScript: Classes

A demonstation on how to use javascript Classes
